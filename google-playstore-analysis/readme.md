# Google Play Store Data Cleaning

## Overview
Project ini membersihkan dataset Google Play Store menggunakan SQL.
Fokus pada:
- Handling missing values
- Standardisasi tipe data
- Cleaning kolom numerik & tanggal
- Normalisasi format versi aplikasi
- Penghapusan duplikasi

## Tools
- MySQL 8
- SQL Data Cleaning

## Files
- cleaned-data.csv — dataset final
- query-cleaning.sql — seluruh query cleaning
